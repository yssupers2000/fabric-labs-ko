# Microsoft Fabric 시작하기

이 repo에는 Microsoft Learn의 [Microsoft Fabric 시작하기](https://aka.ms/learn-fabric) 학습 경로에 있는 연습을 완료하는 데 필요한 지침과 자산이 포함되어 있습니다.

### 이슈 보고

연습에서 문제가 발생하는 경우, 이 repo에 **이슈**로 보고해 주십시오.

> **참고**: 연습 내용과 Fabric 서비스 자체에 대한 피드백을 환영하지만, 이 repo에서는 연습 내용과 관련된 이슈만 지원할 수 있습니다. Microsoft Fabric 서비스 자체와 관련된 기술적 이슈 또는 피드백은 [https://support.fabric.microsoft.com/support/](https://support.fabric.microsoft.com/support/)에 보고해 주십시오.
